# 調査結果（spomapjp.com）

## 概要
公開サイト（https://www.spomapjp.com/）と主要ページを確認し、既存の収益導線と改善点を整理。

## 確認したページ
- `/`
- `/gyms`
- `/search`
- `/map`
- `/gyms/nearby`
- `/about`
- `/contact`
- `/privacy`
- `/terms`
- ジム詳細（sitemap から抽出した 1 件）

## 現状の観測結果
- ジム詳細ページに「おすすめグッズ」（アフィ）導線あり。
  - 楽天アフィリンクが表示されることを確認。
  - Amazon アフィリンクは表示されていない（設定なし or 未表示）。
- Google AdSense のスクリプトは読み込まれているが、詳細ページの広告枠 `slotId` が未設定。
  - `YOUR_SLOT_ID_HERE` のままのため、広告表示が無効。
- ブランド表記に不一致がある。
  - ジム詳細メタの title が `IRON MAP` になっている。
  - プライバシーポリシーのタイトルが「公営ジム検索くん」。
  - 公式名称「SPOMAP」との統一が必要。
- 詳細ページには以下の導線がある。
  - 公式サイトリンク
  - お気に入り
  - 問題報告

## マネタイズに向けた機能改善（優先順）
1. **AdSense 本稼働**
   - `slotId` を実値に変更し広告を表示させる。
   - CLS 対策として固定高さの広告枠/遅延表示を実装。
2. **送客計測の実装**
   - 公式サイト/アフィリンクのクリック計測を追加。
   - 検索 → 詳細 → 外部遷移のファネルを可視化。
3. **アフィ拡張**
   - Amazon タグ設定を追加し 2 面出し。
   - 設備情報に連動した商材推薦を強化。
4. **オーナー向け課金（Claimed Listing）**
   - 施設オーナーが営業時間/料金/写真/特典を更新できる仕組みを有料で提供。
   - 上位表示/バッジなど露出強化とセットで月額化。
5. **リード課金（問い合わせ/見学予約）**
   - 施設向け問い合わせフォームを用意し、1 件課金 or 月額課金。
6. **B2B データ提供**
   - API キーと利用量課金を用意し、事業者向けにデータ提供。
7. **ユーザー課金（Pro）**
   - 保存検索、通知（設備追加/営業時間変更など）を有料機能として提供。
8. **データ鮮度運用の強化**
   - 差分検知・再スクレイプ・ユーザー報告の反映で品質向上。

## マネタイズに向けた処理提案（実装観点）
- **クリック/コンバージョン計測の永続化**: 外部遷移（公式/アフィ）や広告表示/クリックをイベントログとして保存し、ダッシュボードで CTR/CTR×CV を可視化。
- **広告枠の設定管理**: `slotId` を環境変数 or 管理画面で差し替え可能にし、ページ別に A/B テスト可能な構成にする。
- **アフィリンク生成の統一**: 楽天/Amazon のリンク生成を共通関数化し、商材/カテゴリ単位で出し分けできるよう拡張。
- **スポンサー枠の順位制御**: 検索/地図結果の上位枠に「PR」扱いを差し込むためのスコア補正と表示ラベルを追加。
- **オーナー認証フロー**: 申請 → 審査 → `owner_verified` 付与 → 編集権限付与の管理フローと監査ログを追加。
- **問い合わせ処理**: 施設への問い合わせ送信 API と通知（メール/Slack/Discord など）を追加し、スパム対策とレート制限を実装。
- **課金基盤の準備**: 認証（OAuth）と課金（Stripe 等）の最小導入、権限チェックで Pro 機能を制御。
- **鮮度更新パイプライン**: 再スクレイプの定期実行、差分検知、ユーザー報告の反映をワークフロー化。
